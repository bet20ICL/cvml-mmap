# cvml-mmap

Personal knowledge base (Obsidian vault) for computer vision, SLAM, machine learning, and deep learning concepts.

## Areas of Focus

- **Vision-Language-Action (VLA) Models** — perception-to-action systems built on vision-language pretraining
- **BEV-based Localisation** — bird's eye view representations for metric localisation (e.g. OrienterNet)
- **Shared Foundations** — contrastive learning, transformers, visual representations

## Structure

| Folder | Purpose |
|--------|---------|
| `MOC/` | Maps of Content — index notes linking related concepts |
| `Concepts/` | Foundational ideas: losses, representations, training paradigms |
| `Models/` | Specific architectures and systems |
| `Tasks/` | Problem definitions and task-level surveys |

## Usage

Open this folder as an Obsidian vault. The graph view will show `Contrastive Learning` as the central hub linking the two main research threads.
