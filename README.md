# cvml-mmap

A Obsidian vault mindmap for understanding CV/ML concepts. 

I started this since I find it hard to keep up with all the latest advancements in the field. 

I noticed:
- models often had a "history" (e.g. PointTransformer -> PT2 -> PT3 -> Sonata). Starting from the oldest model and incrementally building up understanding was often more approachable than jumping straight to the latest development.
- domains which often seemed quite different (e.g. VLAs and BEV Localisation) often have common underlying principles (e.g. contrastive learning used both in SigLIP encoder for VLAs and training OrienterNet model)

These points above motivated me to organise the knowledge into this mindmap!
